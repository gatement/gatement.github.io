Android - User PhoneGap to build Android application

1. install Android SDK (add PATH to /tools and /platform-tools)
2. install Ant (add PATH)
3. download PhoneGap and unzip it

4. go to the PhoneGap/lib/android
(5. bin/create -h)
6. bin/create

7. ant debug/release/installd/installr/debug install/release install/clean
