Oracle - Foundation

Oracle Net is in charge of client and server connections.

user app / SQLPLUS –> local Oracle Net –> TCP/IP(Internet) –> Listerner(IP/Port/service_name)[server Oracle Net] –> service –> DB

1. config TNS names – service_alias(optional)
- edit file “installed_dir/network/admin/tnsnames.ora”
- or use GUI Oracle “Net Manager/Service Naming”

2. open SQLPLUS

sqlplus /nolog

connect

CONN user/pwd@service_alias

or 

CONN user/pwd@//server_ip:port/service_name

3. list all tables in the databse

select object_name from user_objects where object_type='TABLE';
