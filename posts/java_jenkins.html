3. Useful System Properties (Add to JAVA_ARGS)

  • -Dhudson.udp=-1 (disable UDP broadcast/multicast at defaupt port 33848)
  • -Dhudson.DNSMultiCast.disabled=true (disable DNS multicast at default port 5353)
    JAVA_ARGS="-Djava.awt.headless=true -Dhudson.DNSMultiCast.disabled=true -Dhudson.udp=-1"
