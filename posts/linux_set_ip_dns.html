Linux - Set IP/DNS

1. Set static ip
Edit /etc/sysconfg/network-scripts/ifcfg-eth0

DEVICE="eth0"
BOOTPROTO="static"
HWADDR="xx:xx:xx:xx:xx:xx"
ONBOOT="yes"
UUID="xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxx"
IPADDR=192.168.1.102
NETMASK=255.255.255.0
GATEWAY=192.168.1.1

2. Set DNS
Edit /etc/resolv.conf

nameserver 8.8.8.8 
nameserver 8.8.4.4

3. Set gateway for the machine (optional)
Edit /etc/sysconfig/network
