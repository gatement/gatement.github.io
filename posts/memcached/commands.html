memcached commands

  • get into command line: 
    telnet localhost 11211
  • show settings 
    stats settings
  • list items 
    stats items
  • list item caches 
    # stats cachedump {item_number} 0
# the last 0 stands for list all caches in the item
stats cachedump 7 0
  • get cache 
    get {cache_key}
  • clear all caches 
    flush_all
  • set cache 
    # one 0 is flags
# another 0 is expiration
# {len} is content byte size
set 0 0 {len}
