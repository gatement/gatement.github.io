memcached commands

  • get into command line: 
    telnet localhost 11211
  • show settings 
    stats settings
  • list items 
    stats items
  • list item caches 
    # the last 0 stands for list all caches in the item
# stats cachedump {item_number} 0
stats cachedump 7 0
  • get cache 
    get {cache_key}
