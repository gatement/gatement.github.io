BitTorrent - Deluge Setup

http://deluge-torrent.org/

## install
yum install deluge-daemon
yum install deluge-console
yum install deluge-web

## create auth file
echo “username:password:level” >> /root/deluge/auth

## start deluge daemon
deluged -c /root/deluge -p 58846

## config allowing remote client connect
deluge-console -c /root/deluge
config -s allow_remote True
config allow_remote
exit

## restart deluge daemon
pkill deluged
deluged -c /root/deluge -p 58846

## start deluge web daemon
deluge-web –fork –ssl -c /root/deluge

## open deluge web client in browser (can be remotely)

http://[server]:8112

Default password is “deluge”
