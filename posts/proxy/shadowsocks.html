Shadowsocks

  • install 
        pip install shadowsocks
  • config 
        # vi /etc/shadowsocks.json
    {
        "server": "0.0.0.0",
        "server_port": 8388,
        "password": "mypassword",
        "timeout": 300,
        "method": "aes-256-cfb",
        "fast_open": false
    }
  • start 
        ssserver -c /etc/shadowsocks.json -d start # log to /var/log/shadowsocks.log
