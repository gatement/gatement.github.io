vim: setup development env

1. install pathogen
https://github.com/tpope/vim-pathogen 
mkdir -p ~/.vim/autoload
mkdir -p ~/.vim/bundle
cd ~/.vim/bundle
wget https://raw.githubusercontent.com/tpope/vim-pathogen/master/autoload/pathogen.vim
edit ~/.vim/vimrc:
    call pathogen#infect()
    call pathogen#helptags()
(replace "~/.vim" with "~\vimfiles" in windows)
(it will manage "runtimepath" and auto ":helptags")

2. install plugins
https://github.com/jimenezrick/vimerl (vimerl)
https://github.com/kien/ctrlp.vim (ctrlp)
https://github.com/scrooloose/nerdtree (nerdtree)
https://github.com/jistr/vim-nerdtree-tabs (nerdtree-tabs)
http://www.vim.org/scripts/script.php?script_id=273 (taglist) 
cd ~/.vim/bundle
git clone ...

(taglist only):
*nix: yum/apt-get/brew install ctags
windows: http://sourceforge.net/projects/ctags/ (config it to the PATH)

3. install color scheme
https://github.com/tpope/vim-vividchalk 
mkdir -p ~/.vim/colors
cd ~/.vim/colors
wget https://raw.githubusercontent.com/tpope/vim-vividchalk/master/colors/vividchalk.vim

refer to http://blog.erlware.org/2013/09/09/how-to-use-vim-for-erlang-development/
